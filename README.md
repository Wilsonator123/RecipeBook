# Recipe Book

S - I have recipes from many different websites, videos and book and keeping track of all of the different recipes is difficult

T - Centralise all recipes into one solution to allow users to store and read through all their recipes from a variety of sources

A - Build a website using NextJS and C# which allows users to create custom recipes or insert a link to a website or video where the data is scraped

R - Built a well-designed website which met all criteria and recieved high marks as a result

## Tech-stack

NestJS, JavaScript, C#, MongoDB, Terraform
